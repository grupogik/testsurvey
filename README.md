# testsurvey
Es una aplicación por la cual los alumnos pueden decir según su punto de vista la dificultad de sus pruebas y en que puntos de la materia se sintieron más vulnerables. El fin de esta app es acercar esta información al profesor para que pueda evaluar futuras mejoras de su curso.

Este repositorio debe ser instalado como plugin local en moodle,
estos documentos deben ir en local/testsurvey
